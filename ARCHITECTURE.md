## 三、Workspace 结构（唯一事实来源）

```

│
```

## 八、关键设计决策

| 决策 | 理由 | 替代方案 |
|------|------|---------|
| 流式 LLM 调用 | 大 prompt 非流式会 90s+ 超时 | 非流式 → 已废弃 |
| 对话压缩 | prompt 膨胀导致超时 | 固定窗口 → 信息丢失 |
| RLIMIT_NPROC 移除 | 按 UID 限制导致 OpenBLAS pthread 失败 | 保留 → 代码执行永远失败 |
| 五层记忆 > 双层 | 准入/提升/衰减/冲突检测 全闭环 | 双层 → 缺精细化控制 |
| 进程沙箱 > 容器 | 个人工具不需要 Docker | 容器 → 过度复杂 |
| Formalism (PlanIR) | 可控 AI 的核心差异化 | ReAct → 约束太弱 |
| 子Agent 不复用主循环 | 独立 system prompt + 独立工具集 | 复用 → 工具边界模糊 |
